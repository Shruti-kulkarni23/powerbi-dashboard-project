# 📊 Power BI Dashboard Project

## 📌 Project Title
Power BI Data Analysis Dashboard

## 📖 Description
This project is a Power BI dashboard created to analyze and visualize data in an interactive way.  
It helps in understanding key insights, trends, and patterns for better decision-making.

## 🎯 Objectives
- To analyze dataset using Power BI
- To create interactive dashboards
- To present insights using charts and visuals
- To support data-driven decision making

## 📂 Project File
- `project.pbix` – Main Power BI file

## 📊 Features
- Interactive Dashboard
- Filters & Slicers
- Data Visualization (Bar charts, Pie charts, Line charts)
- KPI Indicators
- User-friendly layout

## 🛠 Tools & Technologies
- Power BI
- Excel / CSV Dataset
- Data Cleaning
- Data Visualization
